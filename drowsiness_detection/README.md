# [Drowsiness detection with OpenCV](https://pyimagesearch.com/2017/05/08/drowsiness-detection-opencv)

## Usage

```shell
python detect_drowsiness.py -p <path to facial landmark predictor file> --alarm <path to wav file>
```

## Installing `playsound`

To install `playsound`, I had to install `wheel` first.
For details, see [the issue](https://github.com/TaylorSMarks/playsound/issues/150) at original repository.
